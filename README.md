# MDX 静态博客

这是一个使用 Next.js 和 Contentlayer 构建的静态博客项目。

## 功能

- 使用 MDX 编写博客文章
- 支持暗黑模式
- 响应式设计
- 使用 Tailwind CSS 进行样式设计

## 技术栈

- Next.js 14
- React 18
- Contentlayer2
- Tailwind CSS
- TypeScript

## 开始使用

1. 克隆仓库:

```bash
git clone [你的仓库URL]
cd mdx-static-blog
```

2. 安装依赖:

```bash
pnpm i 
```

3. 运行开发服务器:

```bash
pnpm dev
```

4. 打开 [http://localhost:3000](http://localhost:3000) 查看您的网站。

## 构建

要构建生产版本，运行:

```bash
pnpm build
```
